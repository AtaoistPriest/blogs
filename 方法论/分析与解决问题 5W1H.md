# 问题处理模板

## [YYYY-MM-DD] 问题简述
- **What**：系统在连续运行72小时后发生Hard Fault
- **Why**：怀疑是内存泄漏导致的堆溢出
- **Who**：发现者：我；协助者：李工
- **When**：压力测试第3天上午10:23
- **Where**：在网络数据包处理模块中
- **How（排查过程）**：
  1. 使用`freeRTOS heap`监控工具，发现可用堆内存持续减少
  2. 排查所有`malloc`/`free`调用，发现某处异常处理分支漏了`free`
  3. 修复后，连续运行7天无异常
- **经验教训**：
  - 对于动态内存分配，必须确保所有路径都有正确释放
  - 建议使用静态分配或内存池替代频繁的动态分配

# What（是什么）

* 问题或任务的具体内容是什么？需要完成什么目标？

* 当前问题现象是什么：系统崩溃？功能异常？可见信号？有日志输出吗？
* 问题复现步骤？复现概率？
* 影响范围？

# Why（为什么）

* 为什么会出现这个问题？

* 为什么要做这件事？不做的后果是什么？

* 背后的原因和动机是什么？

# Who（谁）

* 谁是相关人员？（参与者、负责人、影响对象等）

* 谁应该承担责任？

* 谁能提供帮助或资源？

# When（何时）

* 问题何时发生？

* 计划何时开始、何时完成？

* 有没有关键时间节点或截止日期？

# Where（何地）

* 问题发生在什么地点或场景？

* 相关的环境、条件是什么？

* 在哪里执行任务更合适？

# How（如何）

* 如何解决这个问题？基于日志排除、对比法、使用工具排除、

* 如何制定解决方案？是否会引入新的问题？是否方便维护？是否可兼容？不会影响别的模块与功能？是否影响性能占用更多资源？

* 如何有效验证？
